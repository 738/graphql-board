# graphql-board DEMO

## Skill Stack

- React
- create-react-app
- Typescript
- GraphQL
- apollo-server
- apollo-client

## Running

### Server:

```bash
cd server
npm i
npm run dev
```

- HTTP GraphQL endpoint is available at: http://localhost:4000/

### Client:

```bash
cd client
npm i
npm run start
```

- URL: http://localhost:3000
